# Overflow

**Enum** | `orca.UIKit`

Content overflow handling strategies

## Constants

| Name | Description |
|------|-------------|
| `Visible` | Content overflows container bounds |
| `Hidden` | Clip overflowing content |
| `Clip` | Hard clipping without scrollbars |
| `Scroll` | Show scrollbars for overflow |
| `Auto` | Automatic scrollbar behavior |
