# UnderlineShorthand

**Struct** | `orca.UIKit`

Defines shorthand underline properties (offset, width, and color).

## Fields

| Field | Type | Description |
|-------|------|-------------|
| `Offset` | `float` | Vertical offset of the underline from the baseline. |
| `Width` | `float` | Line thickness of the underline. |
| `Color` | `color` | Color of the underline. |
