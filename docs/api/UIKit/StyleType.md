# StyleType

**Enum** | `orca.UIKit`

Specifies the classification of a style and how it is applied to components.

## Overview

StyleType determines the purpose and scope of a style definition. Generic styles apply globally to all matching targets, while specialized types define overrides or theme-specific variants for more controlled styling behavior.

## Constants

| Name | Description |
|------|-------------|
| `Generic` | Base style applied to all components of the specified target type. Serves as a default visual definition. |
| `Named` | Style associated with a specific theme or visual skin. Used to switch appearance dynamically based on active theme. |
