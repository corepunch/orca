# Page

**Class** | `orca.UIKit` | extends [Node2D](Node2D.md)

Represents a single page within a document or UI container.

## Overview

Used to store navigation related data while all the functionality comes from PageHost

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `Title` | `fixed` | The title of the page. |
| `Path` | `fixed` | The URL path of the page, like "/about". |
| `Transition` | `float` | The transition progress of the page. |
