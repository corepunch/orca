# PageHost

**Class** | `orca.UIKit` | extends [Page](Page.md)

Container that manages multiple pages and navigation between them.

## Overview

Provides functionality for navigating between different Page instances, including maintaining a navigation stack, handling back/forward navigation, and managing page transitions.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `ActivePage` | `Page` | The currently active page. |
