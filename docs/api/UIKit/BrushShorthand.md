# BrushShorthand

**Struct** | `orca.UIKit`

Represents a generic brush used for filling UI elements.

## Overview

A brush can be used as a background, foreground, or any paintable property. It supports multiple types of fills; each field corresponds to one type of fill.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| `Color` | `color` | Solid color fill. |
| `Image` | `Texture` | Texture/image fill. |
| `Material` | `Material` | Custom material or shader fill. Advanced fill using shader/material settings for effects beyond simple color or image. |
