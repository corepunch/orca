# Cinematic

**Class** | `orca.UIKit` | extends [Node2D](Node2D.md)

Video or animation playback view for media content

## Overview

Handles playback of video files or frame-based animations with timing control and fade effects.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `FileName` | `fixed` | Path to media file for playback |
| `FrameRate` | `float` | Playback frame rate (frames per second) |
| `NumFrames` | `float` | Total number of frames in the media |
| `FadeOut` | `float` | Fade-out duration in seconds |
