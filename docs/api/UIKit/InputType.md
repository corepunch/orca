# InputType

**Enum** | `orca.UIKit`

Input control behavior types for different data entry needs

## Constants

| Name | Description |
|------|-------------|
| `Text` | Single or multi-line text input  |
| `Password` | Masked password input for security  |
| `Number` | Numeric input with validation  |
| `Checkbox` | Boolean checkbox control  |
| `Radio` | Radio button for exclusive selection  |
