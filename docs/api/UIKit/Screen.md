# Screen

**Class** | `orca.UIKit` | extends [Node2D](Node2D.md)

Top-level container representing a display surface or window

## Overview

Provides the root rendering context for UI hierarchies. Manages screen-level operations like clearing and render target management.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `ClearColor` | `color` | Background clear color for the screen |
| `ResizeMode` | `ResizeMode` | Specifies how the window can be resized or displayed |
