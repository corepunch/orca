# ColorBrush

**Class** | `orca.UIKit` | extends [Brush](Brush.md)

Brush that fills an area with a solid color.

## Overview

ColorBrush applies a uniform color to a drawable surface. It is typically used for flat-colored elements such as backgrounds, highlights, or overlays where no texture or material is required.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `Color` | `color` | The color value used to fill the target area. |
