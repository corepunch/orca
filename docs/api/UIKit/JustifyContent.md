# JustifyContent

**Enum** | `orca.UIKit`

Main-axis content distribution in flexbox-style layouts

## Constants

| Name | Description |
|------|-------------|
| `Start` | Pack items to start |
| `End` | Pack items to end |
| `Center` | Center items |
| `SpaceBetween` | Distribute with space between |
| `SpaceAround` | Distribute with space around |
| `SpaceEvenly` | Distribute with even spacing |
