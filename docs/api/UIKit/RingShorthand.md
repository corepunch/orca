# RingShorthand

**Struct** | `orca.UIKit`

Decorative ring/outline effect around elements

## Overview

Creates an outline ring drawn around an element's outer edge. Useful for accessibility and visual feedback such as focus indicators.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| `Offset` | `float` | Distance from element edge to ring |
| `Width` | `float` | Ring thickness in pixels |
| `Color` | `color` | Ring color and opacity |
