# AlignmentShorthand

**Struct** | `orca.UIKit`

Axis-specific alignment parameters.

## Overview

Stores per-axis alignment information (e.g., Horizontal, Vertical, Depth). Used by Node and its derived classes to control alignment.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| `Axis` | `int[3]` | Alignment of the element within its parent (similar to HorizontalAlignment/VerticalAlignment in XAML). |
