# Box3Field

**Enum** | `orca.UIKit`

Fields of a 3D box, used to describe position and size components

## Constants

| Name | Description |
|------|-------------|
| `X` | X-coordinate of the box origin |
| `Y` | Y-coordinate of the box origin |
| `Z` | Z-coordinate of the box origin |
| `Width` | Width of the box along the X axis |
| `Height` | Height of the box along the Y axis |
| `Depth` | Depth of the box along the Z axis |
