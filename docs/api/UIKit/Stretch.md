# Stretch

**Enum** | `orca.UIKit`

Specifies how the content of an element is resized to fill its layout area

## Constants

| Name | Description |
|------|-------------|
| `Uniform` | Scale content uniformly to fit within the area, preserving aspect ratio |
| `None` | Do not scale the content; display at its native size |
| `Fill` | Scale content to fill both width and height, ignoring aspect ratio |
| `UniformToFill` | Scale content uniformly to fill the area, preserving aspect ratio but cropping if necessary |
