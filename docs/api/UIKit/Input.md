# Input

**Class** | `orca.UIKit` | extends [TextBlock](TextBlock.md)

Interactive text input control with multiple input modes

## Overview

Extends TextBlock to provide editable text capabilities. Supports various input types including text, password, numeric, checkbox, and radio button modes.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `Name` | `fixed` | Input identifier for forms and data binding |
| `Type` | `InputType` | Input behavior type (text, password, etc.) |
| `Cursor` | `int` | Current text cursor position |
| `Multiline` | `bool` | Allow multiple lines of text input |
| `Checked` | `bool` | Checked state for checkbox/radio types |
