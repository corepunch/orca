# EventTrigger

**Class** | `orca.UIKit` | extends [Trigger](Trigger.md)

Responds to routed UI events (mouse, keyboard, etc.)

## Overview

Listens for specific routed events and can execute actions in response. Supports the standard UI event bubbling system.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `RoutedEvent` | `fixed` | Name of the routed event to handle (e.g., "MouseDown", "KeyPress") |
