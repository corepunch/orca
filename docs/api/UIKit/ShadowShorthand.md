# ShadowShorthand

**Struct** | `orca.UIKit`

Drop shadow effect configuration

## Overview

Defines a drop shadow with offset, blur, spread, and color.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| `Offset` | `vec2` | Shadow offset from element (x, y) |
| `BlurRadius` | `float` | Blur amount for shadow softness |
| `SpreadRadius` | `float` | Shadow expansion beyond element bounds |
| `Color` | `color` | Shadow color and opacity |
