# Label

**Class** | `orca.UIKit` | extends [TextBlock](TextBlock.md)

Text label with optional association to form controls

## Overview

Displays static text that can be associated with input controls for accessibility. Clicking the label can transfer focus to associated control.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `For` | `fixed` | Identifier of associated form control |
