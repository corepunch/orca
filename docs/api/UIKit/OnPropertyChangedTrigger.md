# OnPropertyChangedTrigger

**Class** | `orca.UIKit` | extends [Trigger](Trigger.md)

Responds to property value changes on specified nodes

## Overview

Monitors a named property on a source node and triggers actions when the property value changes. Useful for reactive UI updates.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `SourceNode` | `fixed` | Name or identifier of the node to monitor |
| `Property` | `fixed` | Property name to watch for changes |
