# Handler

**Class** | `orca.UIKit`

Executes custom functions when triggered by specified conditions

## Overview

More flexible than Setter - can call arbitrary functions on target objects rather than just setting properties.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `Trigger` | `Trigger` | Triggering condition or state image |
| `Target` | `Node` | Target object to call function on |
| `Function` | `fixed` | Function name to execute on target |
