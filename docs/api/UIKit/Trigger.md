# Trigger

**Class** | `orca.UIKit`

Base class for event-driven actions within the UI system

## Overview

Triggers respond to various conditions and can execute actions like property changes or method calls. Part of the reactive UI behavior system.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `Property` | `fixed` | Target property name to monitor or modify |
| `Value` | `int` | Associated value for the trigger condition |
