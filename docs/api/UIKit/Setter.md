# Setter

**Class** | `orca.UIKit`

Applies property values when triggered by specified conditions

## Overview

Part of the trigger system - sets target properties to specific values when the associated trigger condition is met.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `Trigger` | `Trigger` | Triggering condition or state image |
| `Property` | `fixed` | Target property name to set |
| `Value` | `fixed` | Value to apply to the property |
