# BorderStyle

**Enum** | `orca.UIKit`

Visual styles for element borders

## Constants

| Name | Description |
|------|-------------|
| `None` | No border |
| `Hidden` | Hidden border (takes space but invisible) |
| `Dotted` | Dotted border pattern |
| `Dashed` | Dashed border pattern |
| `Solid` | Solid continuous border |
| `Double` | Double line border |
| `Groove` | 3D grooved border effect |
| `Ridge` | 3D ridged border effect |
| `Inset` | 3D inset border effect |
| `Outset` | 3D outset border effect |
