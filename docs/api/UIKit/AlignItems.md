# AlignItems

**Enum** | `orca.UIKit`

Cross-axis alignment in flexbox-style layout containers

## Constants

| Name | Description |
|------|-------------|
| `Start` | Align to start of cross axis |
| `Center` | Center on cross axis |
| `End` | Align to end of cross axis |
| `Stretch` | Stretch across cross axis |
| `Baseline` | Align to text baseline |
