# RotationOrder

**Enum** | `orca.geometry`

Euler angle rotation order enumeration

## Overview

Specifies the order in which rotations are applied around the X, Y, and Z axes. Different orders produce different final orientations for the same euler angles.

## Constants

| Name | Description |
|------|-------------|
| `XYZ` | Rotate around X axis first, then Y, then Z |
| `XZY` | Rotate around X axis first, then Z, then Y |
| `YZX` | Rotate around Y axis first, then Z, then X |
| `YXZ` | Rotate around Y axis first, then X, then Z |
| `ZXY` | Rotate around Z axis first, then X, then Y |
| `ZYX` | Rotate around Z axis first, then Y, then X |
