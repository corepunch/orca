# Size

**Struct** | `orca.geometry`

2D size structure

## Overview

Represents the dimensions of a 2D object using width and height. Provides utility methods for scaling and aspect ratio calculations.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| `width` | `float` | Width of the object |
| `height` | `float` | Height of the object |
