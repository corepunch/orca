# sphere3

**Struct** | `orca.geometry`

3D sphere representation

## Overview

Represents a sphere in 3D space defined by center point and radius. Used for collision detection, bounding volumes, and spatial queries.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| `center` | `vec3` | Center point of the sphere |
| `radius` | `float` | Radius of the sphere |
