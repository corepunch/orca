# SKView

**Class** | `orca.SpriteKit` | extends [Node2D](Node2D.md)

2D viewport that renders an SKScene into the UI hierarchy

## Overview

Acts as the bridge between the 2D UI system and the sprite scene hierarchy. Renders the attached SKScene using a 2D orthographic projection into its Node2D bounds.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `ReferenceWidth` | `float` | Reference width for the sprite coordinate system |
| `ReferenceHeight` | `float` | Reference height for the sprite coordinate system |
| `Scene` | `fixed` | Name of SKScene to render |
