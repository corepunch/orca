# SKLabelNode

**Class** | `orca.SpriteKit` | extends [SKNode](SKNode.md)

2D text label node for displaying text in sprite space

## Overview

Renders text as a 2D sprite node that participates in the sprite scene hierarchy. Inherits text styling and layout from the TextBlockConcept.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `Color` | `color` | Color of the text |
