# CullMode

**Enum** | `orca.SceneKit`

Face culling modes to optimize rendering by discarding non-visible triangle faces

## Constants

| Name | Description |
|------|-------------|
| `None` | No face culling - render both front and back faces |
| `Back` | Cull back-facing triangles (standard for solid objects) |
| `Front` | Cull front-facing triangles (useful for interior views or special effects) |
