# Model3D

**Class** | `orca.SceneKit` | extends [Node3D](Node3D.md)

3D model renderer that displays mesh geometry with materials

## Overview

Core component for rendering 3D models. Combines a mesh (geometry) with a material (shading) to create visible 3D objects in the scene.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `Mesh` | `Mesh` | 3D mesh geometry defining the shape of the model |
| `Material` | `Material` | Material defining surface appearance, textures, and shading properties |
