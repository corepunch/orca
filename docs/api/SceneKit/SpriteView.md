# SpriteView

**Class** | `orca.SceneKit` | extends [Node3D](Node3D.md)

Billboard sprite renderer for 2D images in 3D space

## Overview

Displays 2D images as billboards that automatically face the camera. Useful for particle effects, UI elements in 3D, icons, and other 2D content that needs 3D positioning.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `Image` | `Texture` | 2D image texture to display on the billboard sprite |
| `Bounds` | `vec4` | Texture coordinates and sizing (X=U, Y=V, Z=width, W=height) |
