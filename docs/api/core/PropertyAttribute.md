## Constants

| Name | Description |
|------|-------------|
| `WholeProperty` | Default binding to the whole property |
| `ColorR` | Bind to the red channel of a color property |
| `ColorG` | Bind to the green channel of a color property |
| `ColorB` | Bind to the blue channel of a color property |
| `ColorA` | Bind to the alpha channel of a color property |
| `VectorX` | Bind to the X component of a vector property |
| `VectorY` | Bind to the Y component of a vector property |
| `VectorZ` | Bind to the Z component of a vector property |
| `VectorW` | Bind to the W component of a vector property |
