Enum

RotationOrder

Euler angle rotation order enumeration

Overview

Specifies the order in which rotations are applied around the X, Y, and Z axes. Different orders produce different final orientations for the same euler angles.

Constants

case XYZ
Rotate around X axis first, then Y, then Z
case XZY
Rotate around X axis first, then Z, then Y
case YZX
Rotate around Y axis first, then Z, then X
case YXZ
Rotate around Y axis first, then X, then Z
case ZXY
Rotate around Z axis first, then X, then Y
case ZYX
Rotate around Z axis first, then Y, then X
