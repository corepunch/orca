MipmapMode

Constants

case Base
Only the base (highest-resolution) mip level is used. Fastest, but can cause aliasing when textures are minified.
case Nearest
Selects the nearest mip level based on the distance, without blending between levels. Produces sharper but less smooth transitions.
case Linear
Interpolates between two nearest mip levels for smoother transitions. Common default for high-quality rendering.
case Trilinear
Performs linear filtering across both texel and mip levels, providing the smoothest possible texture transitions (slightly slower).
